# Izakaya_FrontEnd

## 專案介紹:
使用Vue3+Vite ，搭建 居酒屋網站

## 使用技術
1. VuetifyUI
2. Pinia
4. HTML / CSS / JavaScript
5. vue-Router
6. axios
7. vue3 Composition API
8. VueUse
9. Https
    
## 主要負責功能:
1. 點餐介面
2. 結帳介面
4. RWD
5. 串接LinePay 金流
